# Website Architecture Plan

This document outlines the information architecture and technical approach for the Kadena Community Website. It follows Issue #3 requirements and will be refined through review.

## Information Architecture
- Global Navigation: Home, About, Roadmap, Contribute, Contact
- Secondary: Code of Conduct, Wiki entry points

## Pages
- Home: Welcome, quick links
- About: Mission, vision
- Roadmap: Milestones
- Contribute: How to get involved
- Contact: Contact methods

## Footer
Links, social, license/attribution.

## Future Expansion
Blog/news, events, localization, search, analytics.

## Technical Notes
- Hosting: GitHub Pages from `main/docs`
- Content sources: Docs and Wiki
- Link strategy: Stable paths; rewrite rules via sync workflow
- Constraints: Protected branches; PR-based wiki sync
